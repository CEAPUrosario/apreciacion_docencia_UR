# Apreciación de la Docencia UR

Bienvenido al repositorio de Apreciación de la Docencia de la Universidad del Rosario.

## 📋 Descripción

Este proyecto contiene recursos, herramientas y documentación relacionados con la evaluación y apreciación de la calidad docente en la Universidad del Rosario.

## 🎯 Objetivos

- Evaluar y mejorar la calidad de la enseñanza
- Proporcionar retroalimentación constructiva a los docentes
- Facilitar el desarrollo profesional docente
- Compartir buenas prácticas educativas

## 📁 Estructura del Proyecto

```
apreciacion_docencia_UR/
├── README.md
├── documentos/          # Documentos y guías
├── recursos/            # Recursos educativos
├── evaluaciones/        # Formularios y evaluaciones
└── resultados/          # Resultados y análisis
```

## 🚀 Primeros Pasos

1. Clona este repositorio:
   ```bash
   git clone https://github.com/CEAPUrosario/apreciacion_docencia_UR.git
   cd apreciacion_docencia_UR
   ```

2. Explora la estructura del proyecto

3. Revisa la documentación disponible
